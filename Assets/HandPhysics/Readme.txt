Hand Physics Controller v1.0.1

Make your own hand simulator game with this physics-based hand controller! Completely ready to use. 

* Smooth control
* Full interaction with Rigidbody objects
* Flexible configuration
* Simple algorithms that does not impair performance 
* Mobile compatibility

High res hand model included (1536 verts, 2048x2048 texture).

---

Check out the example scene in HandPhysics/Example/ folder to see how it works.
